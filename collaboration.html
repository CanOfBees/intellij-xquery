    Setup for development on Linux/Unix/Mac

    • Fork intellij-xquery repository
    • Fetch repository to your machine
      • git clone git@github.com:username/intellij-xquery.git
    • Change directory to newly downloaded directory
      • cd intellij-xquery
    • Download sdk
      • ./gradlew downloadSdk
    • Extract it
      • ./gradlew unzipSdk
    • Build the project
      • ./gradlew build
    • Generate IntelliJ Idea project structure
      • ./gradlew idea
    • Open project
    • In File > Project Structure > Platform Settings > SDKs add IntelliJ Platform Plugin SDK (add JDK first if you haven't already)
    • Select downloaded SDK (sdk sub directory of downloaded plugin sources), your own SDK if you prefer or downloaded source code repository of IntelliJ Community Edition pointing to the correct version tag
    • In Project Settings > Project set newly created plugin SDK as Project SDK
    • Check Plugin development useful links for more information on plugins required for plugin development

    Setup for development on Windows

    • Fork intellij-xquery repository
    • Fetch repository to your machine
      • git clone git@github.com:username/intellij-xquery.git
    • Change directory to newly downloaded directory
      • cd intellij-xquery
    • Change the property in build.gradle file to point to your IntelliJ installation
      • project.ext.userDefinedInstallationPath = 'C:/opt/dev/ideaU'
    • Build the project
      • gradlew build
    • Generate IntelliJ Idea project structure
      • gradlew idea
    • Open project
    • In File > Project Structure > Platform Settings > SDKs add IntelliJ Platform Plugin SDK (add JDK first if you haven't already)
    • Select the location of your IntelliJ Idea installation or downloaded source code repository of IntelliJ Community Edition pointing to the correct version tag
    • In Project Settings > Project set newly created plugin SDK as Project SDK
    • Check Plugin development useful links for more information on plugins required for plugin development
